# CoreJava

CoreJava is a learning project for practicing core Java concepts through small, focused exercises. The goal is to build fluency with Java syntax, standard library usage, object-oriented design, string handling, testing, and problem solving.

This project is intentionally simple: each exercise should help reinforce correct, readable, maintainable Java code. As the project grows, it can be used to practice writing clear APIs, documenting behavior, handling edge cases, and verifying code with unit tests.
